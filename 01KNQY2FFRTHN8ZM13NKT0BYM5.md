# Screen2Note

## Project Overview
Screen2Note is a Windows utility that captures a user-selected screen region, runs offline OCR (Tesseract) on the captured image, and appends the extracted text together with the screenshot to a Markdown note. It is built with C#/.NET 6.0 and works entirely offline, preserving privacy.

## Features
- Global hotkey (`Ctrl + Alt + C`) for instant screen capture
- Transparent overlay for region selection
- Local Tesseract OCR with Korean & English language packs
- Automatic Markdown generation with timestamped entries
- Daily note files (`notes_YYYY-MM-DD.md`) stored in the user’s Documents folder
- JSON-based settings for customizable hotkey, save path, and OCR options
- System-tray resident application with notifications

## Installation
1. **Install Tesseract**
   - Download the Windows installer from the official repository: https://github.com/tesseract-ocr/tesseract
   - Ensure the `tesseract.exe` binary is added to your PATH or placed beside the executable.
   - Install the `eng` and `kor` language packs (place `eng.traineddata` and `kor.traineddata` in the `tessdata` folder).
2. **Download the binary**
   - Grab the latest release ZIP from the GitHub Releases page and extract it to a folder of your choice.
3. **Run** `Screen2Note.exe`. The app will appear in the system tray.

## Build
```bash
# Clone the repository
git clone https://github.com/<your-username>/Screen2Note.git
cd Screen2Note

# Build the project (requires .NET 6 SDK)
dotnet publish -c Release -r win-x64 --self-contained
```
The compiled executable and required DLLs will be placed under `src/Screen2Note/bin/Release/net6.0/win-x64/publish/`.

## Usage
- Press **Ctrl + Alt + C** to start a capture.
- Drag to select the region, release the mouse button.
- The tool saves a temporary PNG, runs Tesseract, and appends the result to the daily Markdown file.
- A notification appears when the note is saved.
