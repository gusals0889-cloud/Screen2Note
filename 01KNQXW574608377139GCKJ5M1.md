# Screen2Note

## Overview
Screen2Note is a lightweight Windows utility that lets you capture any region of your screen, run offline OCR (Tesseract) on the captured image, and automatically append the extracted text, along with the image, to a daily Markdown note. It runs silently in the system tray and is triggered by a configurable global hotkey (default **Ctrl+Alt+C**).

## Features
- **Global hotkey** to instantly start a screen-region capture.
- **Semi-transparent overlay** for intuitive region selection.
- **Offline OCR** using Tesseract OCR.
- **Automatic Markdown generation** with embedded screenshots and timestamps.
- **Per-day notes** (`notes_YYYY-MM-DD.md`) stored in your Documents folder.
- **JSON-based settings** for hotkey, save path, language packs, and more.
- **System tray icon** with a context menu for quick actions.
- **Self-contained .NET 6.0+ build** — no external runtime required.

## Installation
1. Download the latest release from the project’s GitHub Releases page.
2. Extract the archive to a folder of your choice, such as `C:\Screen2Note`.
3. Install [Tesseract OCR](https://github.com/tesseract-ocr/tesseract) and make sure `tesseract.exe` is available in your system `PATH`.
4. Run `Screen2Note.exe`. The app will appear as an icon in the system tray.

## Usage
1. Press **Ctrl+Alt+C** or your configured hotkey to start a capture.
2. Click and drag to select the area you want to capture.
3. Release the mouse button to save the selected region as a temporary image.
4. Tesseract OCR processes the image and extracts the text.
5. The extracted text and screenshot are appended to the current day’s Markdown note.
6. A notification confirms success and shows a preview of the captured text.

## Configuration
Open the settings window from the tray icon and adjust:
- Hotkey combination
- Default notes folder
- OCR language options
- Image format and quality
- Other application preferences

Settings are stored in a `settings.json` file located beside the executable.

## Building from Source
```bash
git clone https://github.com/your-username/Screen2Note.git
cd Screen2Note
dotnet restore
dotnet publish -c Release -r win-x64 --self-contained true -o publish
```

The compiled executable will be available in the `publish` folder.
